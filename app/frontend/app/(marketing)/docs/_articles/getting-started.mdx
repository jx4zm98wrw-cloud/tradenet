<DocsH2 num="01" id="what">What Tradenet does</DocsH2>

Vietnam IP — formally Cục Sở hữu trí tuệ, and previously branded NOIP — publishes a weekly gazette in two sections: <strong>Section A</strong> for newly-published applications (opening their 5-month opposition window), and <strong>Section B</strong> for newly-granted registrations. Each issue runs 80–4,000 pages of dense PDF.

Tradenet ingests every issue the day it publishes, extracts the marks, and gives you four surfaces:

<ul className="docs-list">
  <li><strong>Today</strong> — a digest of what landed since you last checked, anchored on your watchlists</li>
  <li><strong>Search</strong> — text, phonetic, image, and Vienna-code search across the entire corpus</li>
  <li><strong>Detail</strong> — every fact about one mark, including its procedural timeline and opposition window</li>
  <li><strong>Compare</strong> — side-by-side conflict review for 2–3 marks</li>
</ul>

<DocsCallout heading="Tip">
  The fastest way to understand the product is to set up a watchlist for one of your client's brands and let it run for a week. The Monday-morning digest is the moment Tradenet earns its keep.
</DocsCallout>

<DocsH2 num="02" id="how">How the four surfaces fit together</DocsH2>

The product is built around one loop: <strong>watch → find → analyze → act</strong>. A watchlist saves a search. Each new gazette issue re-runs the search and surfaces matches on <strong>Today</strong>. You click into <strong>Detail</strong> to investigate, optionally pull two or three marks into <strong>Compare</strong> for a scorecard, then act — usually by filing opposition before the 5-month window closes.

<DocsH2 num="03" id="first-search">Your first search</DocsH2>

The fastest way to see Tradenet in action: search for a brand you know, then save the search as a watchlist.

<ol style={{ paddingLeft: 22, fontSize: 14.5, color: "var(--ink-2)", lineHeight: 1.8 }}>
  <li>Click <strong>Search</strong> in the top nav (or press <kbd>⌘K</kbd>)</li>
  <li>Switch to the <strong>Phonetic / fuzzy</strong> mode tab</li>
  <li>Type <span className="mono">neur*</span> and pick Class 5 in the rail</li>
  <li>Drag the similarity threshold up to ~75%</li>
  <li>Click <strong>Save as watchlist</strong> in the active-filters bar</li>
</ol>

That's it. Every new gazette issue, Tradenet will run that exact query and surface matches on your Today screen.

<DocsCallout heading="API equivalent">
  Every action you take in the UI maps 1:1 to a REST endpoint. The above is two API calls: <span className="mono">POST /v1/search</span> followed by <span className="mono">POST /v1/watchlists</span>. See the API reference for the full schema.
</DocsCallout>

<DocsCode lang="cURL">{`# Create a watchlist from a phonetic search

  -H "Authorization: Bearer $TRADENET_API_KEY" \\
