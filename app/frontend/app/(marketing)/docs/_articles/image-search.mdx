<DocsH2 num="01" id="None">How it works</DocsH2>

When you upload a specimen, three pipelines run in parallel: a <strong>perceptual hash</strong> for visual similarity, <strong>OCR</strong> to extract the wordmark text, and a <strong>Vienna code classifier</strong> for figurative elements. Results combine all three.

<DocsTable
  headers={["Channel", "What it catches", "Weight (default)"]}
  rows={[
    ["pHash", "Visually identical / near-identical marks", "40%"],
    ["OCR", "Marks that read the same word", "35%"],
    ["Vienna", "Same figurative category (e.g. “stylized V”)", "25%"],
  ]}
/>

<DocsH2 num="02" id="None">What to upload</DocsH2>

<ul className="docs-list">
  <li>The cleanest specimen you have — ideally the one you'd submit to Vietnam IP</li>
  <li>PNG with transparent background works best for visual matching</li>
  <li>For figurative-only marks (no text), the Vienna classifier carries more weight</li>
  <li>For wordmarks, OCR-cleanness matters most — avoid stylized fonts when possible</li>
</ul>

<DocsCallout heading="Pro tip">
  You can upload multiple specimens at once and Tradenet will treat them as a single query. Useful for marks with multiple registered variants (color, B&W, with/without tagline).
</DocsCallout>

<DocsH2 num="03" id="None">Tuning weights</DocsH2>

On the Firm plan and above, you can override the default weights per matter. Pharmaceutical clearance often weights phonetic and visual evenly. Logo conflicts for sports brands weight visual + Vienna heavily, with OCR near zero.
